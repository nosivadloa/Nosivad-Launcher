Nosivad Launcher Install

Recommended:

1. Run:
   NosivadLauncher-Setup-win-x64.exe

2. Follow the installer. It will place the launcher in your local app folder
   and create shortcuts automatically.

Portable option:

1. Extract the full zip to a folder such as:
   C:\Games\NosivadLauncher

2. Run:
   NosivadLauncher.exe

3. Click Launch Game. If the launcher cannot find Legends of Aria Eternal,
   it will ask Steam to launch the base game.

4. Use this login server when connecting:
   nosivad.bounceme.net

5. Join the Discord for testing notes and connection help:
   https://discord.gg/TY5kQ62p9
