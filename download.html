Download the Nosivad Launcher

Players must already own and install Legends of Aria Eternal. The Nosivad launcher does not include the base game client.

Download Installer Portable Zip Join Discord

Install Steps

  1. Install and launch Legends of Aria Eternal at least once through Steam.
  2. Download the Nosivad launcher installer from the latest GitHub release.
  3. Run NosivadLauncher-Setup-win-x64.exe.
  4. Open the launcher and click Launch Game.
  5. At server login, connect to nosivad.bounceme.net.
Important: the download links on this page are placeholders until you replace YOUR-GITHUB-OWNER/YOUR-REPO with your actual GitHub repository path.

Which file should players use?

Installer EXE
Best for most players. Creates shortcuts and uninstall support automatically.
Portable ZIP
Good for testers who prefer to unpack files manually or avoid an installer.

Troubleshooting

  • If the launcher opens but the game does not, confirm the base game is installed through Steam.
  • If Windows warns about the installer, use More info then Run anyway if you trust the source and downloaded it from your official release page.
  • If a player cannot connect, confirm they are using nosivad.bounceme.net and not a local admin API port.
